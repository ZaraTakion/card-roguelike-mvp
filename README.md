# **Card Roguelike MVP**

Um jogo de cartas **deckbuilder roguelike** desenvolvido em **HTML, CSS e JavaScript puro**.
O objetivo deste MVP é criar uma experiência jogável simples: **10 cartas iniciais**, **1 inimigo comum**, **1 mini-chefe**, sistema de **turnos básico**, compra e uso de cartas e interface protótipo.

## **Objetivo do Projeto**

Construir um jogo funcional, responsivo e claro, que sirva como peça de portfólio e demonstração de lógica, organização e design.

## **Funcionalidades do MVP**

• Sistema de turnos
• Mão de cartas
• Uso de cartas (ataque e defesa básicos)
• Estado do jogador
• Inimigo simples com comportamento definido
• Mini-chefe com padrão próprio
• Interface protótipo (wireframe)
• 10 cartas iniciais

## **Tecnologias**

* HTML
* CSS
* JavaScript puro
